# Library Minimal Management System
##  A fully fonctionnal library minimal management system created with Java and JavaFx
##  Description
This is a desktop application built with JavaFX for managing a library system.  
It allows users to manage books, members, and borrowing operations. The admin can add books , add some others 
librarians, users and manage the library 

## Technologies Used
- Java
- JavaFX
- Oracle DataBase
- JDBC

##  Features
- Add, update, delete books
- Manage users/members
- Borrow and return books
- Search books
- Simple GUI with JavaFX

## Usage
1. Download the project or clone it:
git clone https://github.com/Belak17/LibraryManagement

2. Open the project

3. Run the application

4. Use the interface to manage books and users

##  Development Setup

1. Clone the repository:
git clone https://github.com/Belak17/LibraryManagemnt

2. Open the project in IntelliJ IDEA or Eclipse

3. Configure the database in application.properties

4. Build the project using Maven

5. Run the application

## How to tweak this project for your own uses 
Since this is a miniamal system for the management of a library , I'd encourage you to clone this repository and 
rename it to user your onw boilerpate . Its a good starter for those who want to create a fully library 
management system
